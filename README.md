
# 开始

在https://github.com/alibaba/flutter_boost/ 最新的版本上升级至支持空安全，欢迎一起更新维护

## 将FlutterBoost添加到你的Flutter工程依赖中

打开你的工程的pubspec.yaml ，增加以下依赖:

```json
flutter_boost:
    git:
        url: 'https://github.com/sunsxuan/flutter_boost.git'
```
